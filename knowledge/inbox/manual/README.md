# Manual Inbox

This folder is for source material you collected manually and want to turn into structured knowledge entries later.

## Subfolders

- `pending/`: not yet processed
- `processed/`: already turned into a formal knowledge entry
- `review/`: needs manual judgment because of duplicates, uncertainty, or poor quality

## Suggested workflow

1. Put raw Markdown files into `pending/`
2. Review metadata and content quality
3. Convert them into `knowledge/YYYY-MM-DD-title.md`
4. Move the original file to `processed/` or `review/`

# 关于 AI Coding 我的一些总结

> [!quote] 一句话理解
> Vibe Coding 靠感觉，AI Coding 靠人审；加上单元测试，就是把人的"审"变成自动化的"验"，让质量把控从主观走向客观。

## 核心观点

- **Vibe Coding ↔ AI Coding 的区别**：前者靠感觉，后者靠人审。
- **单测的角色**：把人对代码的"审"变成自动化的"验"——这是从主观质量把控走向客观质量把控的关键一步。
- **质量闭环**：人审 + 自动化测试 = 主观判断的"感觉"被可重复执行的客观验证所替代。

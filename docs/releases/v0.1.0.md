# ai-knowledge-vault v0.1.0

## Highlights

- extracted a reusable `kb` skill from the original private vault workflow
- added a clean `knowledge/` template with one sample entry and inbox structure
- replaced hard-coded repo path assumptions with auto-discovery plus `KB_ROOT` fallback
- generalized video ingestion from a private Douyin-only flow to a reusable `video` pipeline
- added English and Chinese documentation for setup, architecture, and transcription

## Included In This Release

- Obsidian-friendly knowledge vault structure
- Claude Code skill under `.claude/skills/kb/`
- concept compilation, health checks, and query reports
- sample knowledge entry and pending inbox example
- DashScope-based optional video and audio transcription workflow

## Notes

- `process-pending` is currently documented as an agent workflow rather than a fully scripted CLI command
- video transcription requires user-provided DashScope credentials and local `ffmpeg`
